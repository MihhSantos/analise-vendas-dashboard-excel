# 📊 Dashboard de Vendas 2024

Projeto de análise de dados desenvolvido em Excel para análise de vendas por região, categoria e vendedor.

## Ferramentas utilizadas

- Excel
- Tabelas Dinâmicas
- Segmentação de dados
- Análise de dados

## Métricas analisadas

- Receita total
- Quantidade vendida
- Ticket médio
- Vendas por região
- Vendas por categoria

- dados/vendas.csv → base de dados

## Objetivo

Demonstrar habilidades de análise de dados e criação de dashboards utilizando Excel.
